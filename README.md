# 🍔 Food Order Application

A full-stack **MERN Stack Food Ordering Web Application** developed as part of the **WSA MERN Stack Internship**. The application enables users to browse restaurants, explore menus, add food items to their cart, place orders, and make secure online payments. It also includes user authentication and AI-powered review analysis to enhance the overall user experience.

## 📌 Features

### User Features

* User Registration and Login
* Secure JWT Authentication
* Browse Restaurants and Menus
* Search Food Items
* Add and Manage Cart
* Place Orders
* Online Payment Integration
* View Order History
* Update User Profile

### Restaurant Features

* Restaurant Listing
* Menu Management
* Food Item Management

### Additional Features

* Coupon Management
* AI Review Analysis
* Responsive User Interface
* Secure REST APIs

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/shravya454/Food_Order_Application.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
npm start
```

### Install Frontend Dependencies

Open a new terminal and run:

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the `backend` folder and configure the required environment variables for MongoDB, JWT, Cloudinary, email service, and payment gateway.
